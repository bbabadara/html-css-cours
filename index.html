🏗️ Éléments, Balises et Attributs

📖 Les fondamentaux du HTML

HTML utilise des balises pour structurer le contenu. La plupart viennent par paires : ouvrante et fermante.

Une balise est un mot-clé entouré de < >.

<nom_balise>Contenu</nom_balise>

Exemple :

<p>Ceci est un paragraphe</p>

Les attributs fournissent des informations supplémentaires : nom="valeur"

<balise attribut="valeur">Contenu</balise>
id - Identifiant unique
class - Classe CSS
src - Source
href - Lien
alt - Texte alternatif
title - Info au survol

Certaines balises n'ont pas de contenu et se ferment elles-mêmes.

<img src="photo.jpg" alt="Description" />
<br />
<hr />
<input type="text" />

⚠️ Erreurs courantes

  • • Oublier de fermer une balise
  • • Mélanger majuscules/minuscules
  • • Oublier les guillemets
  • • Nesting incorrect

Chapitre 3.3•25 min

📦 Le Modèle des Boîtes (Box Model)

📖 Comprendre le Box Model

Chaque élément HTML est une boîte composée de 4 couches :

Content (Contenu)
Padding (Marge intérieure)
Border (Bordure)
Margin (Marge extérieure)

Padding (Marge intérieure)

Espace entre le contenu et la bordure.

padding: 20px;         /* Tous */
padding: 10px 20px;     /* V H */
padding: 5px 10px 15px 20px; /* T R B L */

Margin (Marge extérieure)

Espace entre l'élément et ses voisins.

margin: 20px;         /* Tous */
margin: 10px auto;     /* V H (centré) */
margin-top: 10px;      /* Un seul côté */

Border (Bordure)

La bordure autour de l'élément.

border: 2px solid black;
border-radius: 10px;
border-color: red;
border-width: 1px 2px 3px 4px;

Width & Height

Dimensions de l'élément.

width: 300px;
height: 200px;
max-width: 100%;
box-sizing: border-box;
