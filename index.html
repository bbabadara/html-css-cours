Chapitre 2 • 15 min

🌍 Internet & Le Web

💡 Analogie Simple

Imaginez Internet comme le réseau routier d'un pays. Et le Web comme les voitures qui circulent sur ces routes, transportant des informations.

Définition : Internet est un réseau mondial d'ordinateurs connectés entre eux, permettant la communication et l'échange de données.

Analogie : C'est comme un réseau postal qui permet d'envoyer des lettres (données) partout dans le monde.

Exemples d'utilisation :
  • Email (Gmail, Outlook)
  • Streaming (Netflix, Spotify)
  • Jeux en ligne
  • Téléchargement de fichiers

Définition : Le Web (World Wide Web) est un service d'Internet qui permet d'accéder à des pages via un navigateur.

Analogie : C'est une bibliothèque géante où chaque livre est une page web, et les liens sont les références entre livres.

🖥️ Client

C'est vous ! L'ordinateur/navigateur qui demande et affiche les pages web.

Exemples : Chrome, Firefox, Safari

🖧 Serveur

C'est l'ordinateur distant qui stocke les fichiers du site et les envoie.

Exemples : Apache, Nginx

Flux : Client (demande) → Internet → Serveur (répond) → Internet → Client (affiche)

Chapitre 3 • 25 min

🏗️ HTML - Le Squelette du Web

📖 Définition

HTML (HyperText Markup Language) est le langage de balisage qui définit la structure et le contenu d'une page web.

💡 C'est comme la charpente d'une maison : elle définit les murs, les portes, les fenêtres... mais pas la couleur ni le style !

🔤 Qu'est-ce qu'une balise ?

Une balise est un mot-clé entre crochets qui indique au navigateur comment afficher le contenu.

<nom_balise>Contenu</nom_balise>

✅ Balise avec contenu

<p>Bonjour</p>

S'ouvre, contient du texte, se ferme

🔲 Balise auto-fermante

<br>

Pas de contenu, se ferme sur elle-même

⚠️ Erreurs fréquentes

  • ❌ Oublier de fermer une balise
  • ❌ Fermer dans le mauvais ordre
  • ❌ Mélanger majuscules et minuscules
Balise Rôle Exemple
<h1> Titre principal Le plus grand titre
<h2> à <h6> Sous-titres Hiérarchie des titres
<p> Paragraphe Bloc de texte
<a> Lien hypertexte Cliquez ici
<img> Image Afficher une photo
<ul>, <ol> Listes Listes à puces/numérotées
<div> Bloc generic Conteneur 
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Ma Page</title>
</head>
<body>
  <h1>Bonjour le monde !</h1>
  <p>Bienvenue sur ma première page.</p>
</body>
</html>
<!DOCTYPE html> → Indique que c'est du HTML5
<html lang="fr"> → Balise racine, langue française
<head> → Métadonnées (infos invisibles)
<body> → Contenu visible de la page

Chapitre 4 • 20 min

🎨 CSS - Le Style du Web

📖 Définition

CSS (Cascading Style Sheets) est le langage qui permet de styliser et mise en page les éléments HTML.

💡 C'est comme la décoration intérieure d'une maison : couleurs des murs,样式 des meubles, placement des objets.

sélecteur
{
propriété: valeur;
}

🔍 Explication ligne par ligne

sélecteur → Qui doit être stylisé ? (p, .ma-classe, #mon-id)

propriété → Quelle propriété modifier ? (color, font-size...)

valeur → Quelle valeur appliquer ? (red, 16px...)

p { ... }

Sélectionne tous les paragraphes

.bouton { ... }

Sélectionne les éléments avec class="bouton"

#header { ... }

Sélectionne l'élément avec id="header"

💡 Bonnes pratiques

  • ✓ Utilisez des classes (.ma-classe) pour le style
  • ✓ Utilisez des IDs (#mon-id) pour le JavaScript
  • ✓ Donnez des noms explicites : .btn-primary pas .bp
color: red;

📦 Le Box Model

📖 Définition

Chaque élément HTML est une boîte rectangulaire composée de 4 couches concentriques.

💡 C'est comme envoyer un colis par la poste : le contenu (margin), le papier bulles (padding), la boîte (border), et l'espace entre boîtes (margin).

div {
  /* Espace intérieur */
  padding-top: 20px;
  padding-right: 20px;
  padding-bottom: 20px;
  padding-left: 20px;
  
  /* Raccourci : tous les côtés */
  padding: 20px;
  
  /* Haut/Droite/Bas/Gauche */
  padding: 10px 20px 10px 20px;
  
  /* Bordure */
  border: 2px solid #333;
  
  /* Espace extérieur */
  margin: 10px;
}

⚠️ Erreurs fréquentes

  • ❌ Confondre margin et padding
  • ❌ Margin ne fonctionne pas sur les éléments inline
  • ❌ Marges verticales qui se chevauchent (margin collapse)
  • ❌ Oublier que padding ajoute à la taille de l'élément
Chapitre 6 • 20 min

📐 Flexbox - Disposition Flexible

📖 Définition

Flexbox est un système de disposition qui permet d'aligner et distribuer facilement les éléments dans un conteneur.

💡 C'est comme un chef d'orchestre qui décide comment chaque instrument (élément) doit se placer sur scène.

🎯 Principe

display: flex;
1
2
3

Container = parent avec display: flex
Items = enfants qui se disposent automatiquement

flex-start center flex-end space-between
flex-start center flex-end 
.container {
  display: flex;
  justify-content: center;  /* Horizontal */
  align-items: center;       /* Vertical */
  gap: 20px;                /* Espace entre items */
}
Chapitre 7 • 15 min

📱 Design Responsive

📖 Définition

Le responsive design permet à un site de s'adapter automatiquement à tous les écrans (mobile, tablette, ordinateur).

💡 C'est comme un vêtement extensible qui s'ajuste parfaitement quelle que soit la taille de la personne.

🔍 Meta Viewport

Cette balise est indispensable pour le responsive : 

<meta name="viewport" content="width=device-width, initial-scale=1.0">
/* Par défaut (mobile) */
body {
  background: blue;
}

/* Tablette et plus */
@media (min-width: 768px) {
  body {
    background: green;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  body {
    background: red;
  }
}
