Chapitre 2 • 15 min

🌍 Internet & Le Web

💡 Analogie Simple

Imaginez Internet comme le réseau routier d'un pays. Et le Web comme les voitures qui circulent sur ces routes.

🌐 Internet

Réseau mondial d'ordinateurs connectés permettant la communication et l'échange de données.

Email, Streaming, Jeux en ligne, Fichiers...

🕸️ Le Web

Service d'Internet permettant d'accéder à des pages via un navigateur.

Sites web, Applications en ligne...

💻 Client & Serveur

🖥️ Client = Vous

Navigateur qui demande et affiche (Chrome, Firefox...)

🖧 Serveur = Distant

Machine qui stocke et envoie les fichiers (Apache, Nginx...)

Flux : Client → Internet → Serveur → Internet → Client

Chapitre 3 • 25 min

🏗️ HTML - Le Squelette du Web

📖 Définition

HTML (HyperText Markup Language) est le langage de balisage qui définit la structure et le contenu d'une page web.

💡 C'est comme la charpente d'une maison : murs, portes, fenêtres... mais pas la couleur ni le style !

🔤 Qu'est-ce qu'une balise ?

Un mot-clé entre crochets qui indique au navigateur comment afficher le contenu.

<nom_balise>Contenu</nom_balise>

✅ Balise avec contenu

<p>Bonjour</p>

S'ouvre, contient du texte, se ferme

🔲 Balise auto-fermante

<br>

Pas de contenu

⚠️ Erreurs fréquentes

  • ❌ Oublier de fermer une balise
  • ❌ Fermer dans le mauvais ordre
  • ❌ Mélanger majuscules et minuscules

Chapitre 4 • 20 min

🎨 CSS - Le Style du Web

📖 Définition

CSS (Cascading Style Sheets) permet de styliser et mettre en page les éléments HTML.

💡 C'est la décoration intérieure : couleurs, meubles, placement.

sélecteur
{
propriété: valeur;
}

sélecteur → Qui styliser ? (p, .ma-classe, #mon-id)

propriété → Quelle propriété ? (color, font-size...)

valeur → Quelle valeur ? (red, 16px...)

p { ... }

Sélectionne tous les paragraphes

.bouton { ... }

Sélectionne class="bouton"

#header { ... }

Sélectionne id="header"

💡 Bonnes pratiques

  • ✓ Classes (.ma-classe) pour le style
  • ✓ IDs (#mon-id) pour le JavaScript
  • ✓ Noms explicites : .btn-primary pas .bp

📦 Le Box Model

📖 Définition

Chaque élément HTML est une boîte rectangulaire composée de 4 couches.

💡 Comme envoyer un colis : contenu, papier bulles (padding), boîte (border), espace entre boîtes (margin).

div {
  padding: 20px;           /* Tous les côtés */
  padding: 10px 20px;      /* Haut/Bas Gauche/Droite */
  padding: 5px 10px 15px 20px; /* Haut Droit Bas Gauche */
  
  border: 2px solid #333;
  
  margin: 10px;            /* Tous les côtés */
}

⚠️ Erreurs fréquentes

  • ❌ Confondre margin et padding
  • ❌ Margin ne fonctionne pas sur inline
  • ❌ Marges verticales qui se chevauchent
  • ❌ Padding qui agrandit l'élément
Chapitre 6 • 20 min

📐 Flexbox - Disposition Flexible

📖 Définition

Flexbox permet d'aligner et distribuer facilement les éléments dans un conteneur.

💡 C'est un chef d'orchestre qui décide comment chaque instrument doit se placer.

🎯 Principe

display: flex;
1
2
3

Container = parent avec display: flex
Items = enfants qui se disposent automatiquement

flex-start center flex-end space-between
flex-start center flex-end 
.container {
  display: flex;
  justify-content: center;  /* Horizontal */
  align-items: center;       /* Vertical */
  gap: 20px;                /* Espace entre items */
  flex-wrap: wrap;          /* Retour à la ligne */
}
Chapitre 7 • 15 min

📱 Design Responsive

📖 Définition

Le responsive design permet à un site de s'adapter automatiquement à tous les écrans.

💡 C'est un vêtement extensible qui s'ajuste parfaitement.

🔍 Meta Viewport (Obligatoire !)

<meta name="viewport" content="width=device-width, initial-scale=1.0">
/* Mobile (défaut) */
body { background: blue; }

/* Tablette */
@media (min-width: 768px) {
  body { background: green; }
}

/* Desktop */
@media (min-width: 1024px) {
  body { background: red; }
}

⚡ JavaScript - L'Interactivité

📖 Définition

JavaScript est le langage de programmation qui rend les pages web interactives.

💡 HTML = Structure, CSS = Apparence, JS = Comportement

// Déclarer une variable
let nom = "Marie";      // Peut changer
const age = 25;          // Ne peut pas changer

// Types de données
let texte = "Bonjour";   // String
let nombre = 42;         // Number
let estVrai = true;      // Boolean
let rien = null;         // Null
let indefini;            // Undefined

let = variable modifiable

const = constante (ne change jamais)

// Fonction classique
function direBonjour(nom) {
  return "Bonjour " + nom + " !";
}

// Fonction fléchée
const direAurevoir = (nom) => {
  return "Au revoir " + nom;
};

// Utilisation
direBonjour("Marie");  // "Bonjour Marie !"
// Quand on clique sur un bouton
bouton.addEventListener('click', function() {
  alert('Vous avez cliqué !');
});

// Syntaxe moderne
bouton.addEventListener('click', () => {
  monElement.classList.toggle('actif');
});
